# Legacy Google Sheets Scripts

This directory contains the original Google Apps Script code that SubsTrack was built to replace.

The original setup was a Google Spreadsheet with:
- A **Payments** sheet tracking monthly payments with checkboxes per member
- An **Options** sheet with configuration (price, member list, payment link, etc.)
- Apps Script for sending email reminders via SendGrid
- A custom menu for triggering reminders manually

These files are kept for reference to understand the original data model and business logic.

See `YouTube Premium Working - Payments.csv` and `YouTube Premium Working - Options.csv` in the project root for sample data from the original spreadsheet.
